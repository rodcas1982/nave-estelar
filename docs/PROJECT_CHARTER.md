## 2. Description

A classic space shooter game where the player controls a spaceship, shoots enemies, and avoids asteroids. The goal is to survive as long as possible and get a high score.

**IMPORTANT:** Must be mobile-first with on-screen controls!

## 3. Game Mechanics

### Player
- Spaceship at bottom of screen
- Move left/right with:
  - **Desktop:** Arrow keys, A/D, or mouse
  - **Mobile:** On-screen joystick/buttons (left/right)
- Shoot with:
  - **Desktop:** Spacebar or click
  - **Mobile:** On-screen shoot button
- 3 lives

### Responsive Design
- Full screen canvas
- Touch controls overlay (joystick + buttons)
- Works on mobile, tablet, desktop
- Portrait and landscape support

### Enemies
- Asteroids falling from top
- Enemy ships that shoot back
- Different speeds and sizes

### Scoring
- Points for destroying asteroids
- Points for destroying enemy ships
- Bonus points for combos

### Power-ups (optional)
- Shield
- Triple shot
- Speed boost

## 4. Tech Stack

- **HTML5 Canvas** - Rendering
- **JavaScript** - Game logic
- **CSS** - Styling
- **No frameworks** - Keep it simple

## 5. UI Elements
